# hyoilsfa

효일종합상사가 운영하는 SFA(프랑스 오배수 패키지 펌프) 전문 판매/정보 사이트.
그런포스 전문 사이트(hyoilpump.co.kr)와는 별도 도메인/별도 repo로 운영.

## 프로젝트 목적

- SFA Korea 정식 유통 제품(사니큐빅, 사니콤, 사니프로 등)의 검색 유입 → 신뢰 확보 → 구매 문의 전환
- 타겟: 대기업 영업 채널 외, 검색을 통해 윌로/국산 제품과 SFA를 스스로 비교하는 구매자 전체
  (개인 건축주, 소규모 인테리어업체, 중견시설 담당자 등 규모 불문)

## 포지셔닝

"SFA 정품을 재고로 보유하고, 정가부터 투명하게 공개하며,
어떤 규모의 현장이든 빠르게 대응하는 실속형 구매처"

### 핵심 차별화 (경쟁사 대비)
- 본사(sfa-korea.co.kr): 브랜드/시공사례 스토리텔링 중심, 가격·재고 정보 없음
- 마이펌프(mypump.co.kr): 설계/용역 컨설팅 중심, 상업·신축 대형현장 위주 콘텐츠
- **hyoilsfa**: 정가 공개 + 상담 시 추가 할인, 즉시 재고 확인, 그런포스 대리점 이력 기반 신뢰

## 가격 정책 표기 원칙
- 정가는 투명하게 공개 (경쟁사 대비 차별점)
- 문의/상담 시 별도 할인가 안내를 CTA로 명시 ("정가 문의 시 즉시 상담 할인가 안내")
- 가격을 숨기는 인상을 주지 않도록 주의

## 사이트 구조 (IA)

- 홈 (핵심 메시지 3가지 + 설치 위치별 제품 추천 진입점)
- 왜 SFA인가 (윌로/국산 대비 비교, 원산지 보증, 바닥공사 불필요 원리)
- 제품 (카테고리 → 모델별 상세, 정가 표기)
- 기술자료 (용량 산정 가이드, FAQ — 마이펌프와 겹치지 않는 "소규모/기존 건물" 각도 우선)
- 시공사례
- 회사소개 (효일종합상사 이력, 그런포스 대리점 이력 연결 — 메인 브랜드는 아님)
- 견적/문의

## 브랜드 노출 원칙
- 메인 타이틀/헤드라인은 SFA 제품 중심으로
- "효일종합상사"는 신뢰 근거(회사소개, 푸터, 사업자정보)로만 노출
  → hyoilpump.co.kr과 검색 키워드 자기잠식 방지
- 푸터에 hyoilpump.co.kr 상호링크 (신뢰 신호 + SEO 권위 공유)

## 기술 스택
- GitHub Pages 정적 사이트 (hyoilpump.co.kr과 동일 계정, 별도 repo)
- 도메인/문의 채널(카카오 등)은 오너 결정 예정
